Gets or sets the display name. The display name is normally how a SIP contact should be displayed to the end user.

Gets or sets the URI.
Wildcard
There is a special SIP address which contains a wild card (“*”) which should only be used for a Contact header. Gets or sets a value indicating whether this specific Address is a wild card address.
